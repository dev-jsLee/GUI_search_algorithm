# uv란?
pip처럼 패키지, 라이브러리의 의존성과 가상환경을 관리할 수 있도록 돕는 관리툴이다.
컴파일 언어인 Rust로 제작되어서 필요한 라이브러리를 설치하는 속도가
적게는 3배에서 많게는 10배까지 더 빠르게 설치된다.

# uv 설치
uv는 독립적으로 설치할 수 있지만 python을 설치했다면 pip install uv를 통해서도 사용할 수 있다.
이때 uv를 pip install하는 위치는 컴퓨터에 하는 게 좋다.

# uv 명령어
- uv build
    : pyproject.toml에 적힌 내용을 기반으로 가상환경을 구축하는 명령어
- uv pip~~
    : uv 환경에서 pip 명령어를 사용하는 방법이다. 기존에 pip을 사용하던 유저들도
    이전에 쓰던 명령어를 쓸 수 있기 때문에 적응하기 쉽다.
- uv init [프로젝트명]
    : uv를 사용하는 프로젝트를 만드는 명령어.
    이때 뒤에 공백과 함께 폴더명, 프로젝트명을 작성해주면 해당 이름으로
    폴더가 만들어지고 그 안에 pyproject.toml이 초기작성된다.
